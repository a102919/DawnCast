## 1. 品牌基礎

### 1.1 命名
**DawnCast** — "Dawn"（黎明）+ "Cast"（播送）。
意涵：每早一集英語 podcast，照亮學習者一天的開始。

**品牌關鍵字**：每日、雙人、輕鬆、可信、B1 等級、雙語輔助

### 1.2 定位
> **「每天 3 分鐘，雙人對話帶你熟悉一個英語知識點」**
> 給華語母語、CEFR B1 等級的英語學習者的「聽得懂的英語 podcast」。

### 1.3 價值主張
| 對象 | 痛點 | DawnCast 解方 |
|------|------|---------------|
| 自學者 | 找不到「聽得懂」且「有實質內容」的英語 podcast | B1 等級 + 雙語字幕 + 主題式（科技/文化/生活） |
| 通勤族 | 沒時間看 30 分鐘長節目 | 3 分鐘短集，搭車剛好 |
| 家長 | 孩子的英語教材太無聊 | 主持人有溫度，像聽兩個朋友聊天 |
| 補習班老師 | 缺有趣的補充教材 | SRT/VTT 可下載，可入教材庫 |

### 1.4 核心 KPI（第一年）
- 月活躍訂閱數（MAU subscribers）
- 月留存率（retention）
- 完播率（completion rate，目標 > 75%）
- 推薦係數（referral coefficient，目標 K-factor > 0.3）

## 2. 市場與競品

### 2.1 台灣英語學習市場規模
- 台灣英語學習市場年產值約 NT$2,000 億（含補習班、教材、app、線上課程）
- Podcast 學習者：台灣 Podcast 總聽眾約 600 萬，英語學習類型約 8% 收聽
- 直接競品：VoiceTube、English 4 U、CNN Student News、ESLPod
- 間接競品：Duolingo、Anki、EngVid、YouTube 英語頻道

### 2.2 競品矩陣

| 競品 | 形式 | 長度 | 等級 | 訂閱價 | 雙語字幕 | 我們的差異 |
|------|------|------|------|--------|----------|------------|
| VoiceTube | 影片 + app | 5-15 min | 多元 | 免費/$149/月 | ✅ | DawnCast：純音訊通勤族友善、3 分鐘短、雙人對話 |
| ESLPod | podcast | 20-30 min | A2-B1 | $12/月 | ❌ | DawnCast：3 分鐘、可下載字幕、定價低 |
| Duolingo | app 遊戲 | 5-15 min | A1-A2 | 免費/$84/月 | ❌ | DawnCast：B1 主題式、成人取向 |
| CNN Student News | 影片 podcast | 10 min | B1-B2 | 免費 | ❌ | DawnCast：3 分鐘、CEFR 分級、雙語 |

### 2.3 護城河
1. **內容結構化** — 每集 30s/45s/90s/15s 模板，可 AI 自動生成但有品質一致性
2. **雙語字幕即時生成** — pipeline 已實作，是市面少見
3. **CEFR 等級標註** — 每集標清楚 A2/B1/B2，方便分級學習
4. **本地化** — 華語母語者熟悉的單字挑選、文化主題
5. **可下載資料** — SRT/VTT/JSON 三格式，補習班可入庫

### 3.2 訂閱方案

| 方案 | 月繳 | 季繳（省 10%）| 年繳（省 27%） | 內容 |
|------|------|---------------|----------------|------|
| **Free** | $0 | - | - | 每月 2 集公開、帶字幕 |
| **Basic** | $149 | $399 | $1,299 | 全部集數、雙語字幕下載、學習單字本 |
| **Pro** | $249 | $699 | $2,299 | Basic + 主持人逐句解析、每月 1 次直播 Q&A、主題測驗 |

**年繳折扣目標**：把月留存拉到 70%+，年繳者 LTV 最高。

### 3.3 學習路徑設計

```
Free 試聽 (3 集) → 註冊拿 7 天 Basic 試用 → 月訂閱 → 季/年繳轉換
                                              ↓
                                       完播 + 留言者 → Pro 升級
```

**轉換漏斗目標**：
- 試聽 → 註冊：35%
- 註冊 → 付費：8% (業界 podcast premium 平均)
- 月繳 → 年繳：25%
- Free → Pro：2%

## 4. 供應鏈與技術棧

### 4.1 內容生產 pipeline（已實作 ✅）

```
[Topic] → [Web Search] → [Fact Extraction] → [Vocab Selection] 
       → [Script Writing] → [TTS edge-tts] → [Audio Concat ffmpeg]
       → [Bilingual Subtitle] → [Video MP4 + SRT/VTT/JSON]
```

| 階段 | 工具 | 成本 | 狀態 |
|------|------|------|------|
| 資料檢索 | Brave Search API（web 抓取）| $0 | ✅ POC 完成 |
| 事實萃取 | LLM (本機 + API) | $0-5/月 | ✅ |
| 腳本生成 | LLM | $5-20/月 | ✅ |
| TTS | edge-tts（Guy/Jenny）| **$0** | ✅ POC 完成 |
| 音訊拼接 | ffmpeg | $0 | ✅ |
| 字幕生成 | Python（自寫）| $0 | ✅ |
| 影片燒字幕 | ffmpeg + libass | $0 | ✅ |

**單集製造成本估算**：$0-2（不含人力），比聘請 native speaker 製作 ($200-500/集) 低 99%。

### 4.2 平台與後端架構（V1.1 規劃）

| 元件 | 建議方案 | 月費 | 原因 |
|------|----------|------|------|
| 網站 | Astro + Cloudflare Pages | $0 | 靜態、極便宜、SEO 好 |
| 後端 API | Cloudflare Workers + D1 | $0-5 | 訂閱管理、user auth |
| 付款 | Stripe + 台灣 TapPay | 抽成 2.9% + $0.3 | 國際 + 本地都支援 |
| 音檔/影片託管 | Cloudflare R2 / Bunny.net | $1-5/100GB | 出 1TB 流量的 podcast |
| 訂閱管理 | Stripe Customer Portal | $0 | 免寫後台 |
| Email 自動化 | Resend / Buttondown | $0-20/月 | 試用提醒、續訂通知 |
| Analytics | Plausible / PostHog | $0-9/月 | 隱私友善 |

**固定月成本（100 訂閱者時）**：<$50
**固定月成本（1000 訂閱者時）**：<$200

### 4.3 內容審核流程
- AI 生成 → 自動檢查 CEFR 等級（用 textstat）
- 人工 review → 至少 1 集/週由你或老婆快速聽一遍
- 事實查核 → 高風險主題（醫療/法律/政治）標註警告

## 7. 風險與緩解

| 風險 | 機率 | 影響 | 緩解 |
|------|------|------|------|
| AI TTS 音質不夠好 | 中 | 高 | 雙軌：edge-tts 主、ElevenLabs 備 |
| 內容版權疑慮（AI 改寫）| 中 | 中 | 全部改寫為對話 + 自有事實清單 + 不引用原文 |
| 付費轉換率太低 | 中 | 高 | 強 Free funnel（先 3 集免費）+ 試用期 |
| 競爭對手抄襲 | 低 | 中 | 護城河：雙語字幕 + 結構化模板 + 累積集數 |
| 主持人單一化（永遠 Alex & Sarah）| 低 | 中 | V2 考慮加第三個角色（Diego 西班牙文教英文）|
| 平台演算法變動（小宇宙、IG）| 高 | 中 | 自有官網 RSS 是基本盤，不依賴單一平台 |
