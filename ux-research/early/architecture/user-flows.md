# DawnCast 理想 User Flow（提案）

> pre-product 構想。標出**關鍵決策點**＝產品成敗繫於此的岔路口，做錯就直接複製訪談裡的流失行為。

---

## Flow 1：零阻力首聽（落地即聽，不逼設定）

**目標**：新用戶打開 App 後，在不被要求「設定學習目標 / 選程度 / 建計畫」的情況下，30 秒內聽到第一集且跟得上。直接對抗「查到最後頭很大、什麼都沒下載就去睡」（宜君）與「全英文聽 30 秒就換回台灣 podcast」（佳穎）。

**步驟**
1. 開啟 App → 落地在「今天」，畫面中央就是今日單集，附一行「這集帶你熟什麼」。
2. 一鍵播放，無需註冊、無需選程度（程度由系統先給 B1 預設錨點）。
3. 開始播放，雙語字幕同步捲動，預設可見。
4. 聽到不懂的詞 → 點該詞 → 釋義浮出，播放不中斷。
5. 3 分鐘結束 → 顯示「你這集順手熟了 N 個詞」＋「明天還有一集」的輕量收尾。

**關鍵決策點**
- **要不要先擋一道註冊 / 選程度牆？** → 不要。任何聽前的設定成本都會觸發「頭很大就放棄」。程度用 B1 預設，事後才微調。
- **第一集放什麼？** → 放最能展示「主題對到你 + 難度剛好」的一集（職場場景類），因為首聽 30 秒決定他歸類為「跟不上的全英文」還是「剛剛好」。
- **AI 廉價感的攔截點就在這 30 秒。** 第一集的配音自然度是全產品最高優先，因為「AI 像機器人念課文」是五人共同的棄用焦慮。

## Flow 2：聽到不懂 → 即時查詞 → 之後複習（B1 的核心價值線）

**目標**：讓「聽不懂」從放棄理由變成學習觸點，且查完能回到聽、之後能複習。對抗詩涵「沒字幕完全跟不上、有字幕又變成在看字不在練聽」與佳穎「丟 ChatGPT 問完就沒真的學」。

**步驟**
1. 播放中聽到不懂的字 / 句型 → 點字幕上的該詞。
2. 釋義 + 例句浮出（繁中釋義 + 該詞在本集語境的用法），**播放不停**。
3. 一鍵「加入生字本」。
4. 聽完整集後，可選「快速回放剛剛標記的 3 句」做一次聽力複盤。
5. 之後在「我的收穫 → 生字本」回看，每個詞帶著「當初在哪一集、哪句話」的脈絡。

**關鍵決策點**
- **查詞要不要暫停播放？** → 不暫停（釋義浮層，聽流不斷）。暫停＝把「聽」打斷，違反「像廣播」的心智。
- **字幕預設開還關？** → B1 預設開雙語，但提供「先盲聽、聽完再看字幕對答案」的模式，回應詩涵「看字幕就不算練聽力」的兩難——讓用戶自己選平衡點，而不是替他決定。
- **生字本要不要做成測驗 / 背單字卡？** → 初期不做。一做成測驗就回到「考試壓力」心智，違背「廣播」定位。先做「帶脈絡的回看」就好。

## Flow 3：依主題 / 職場場景挑集（付費開關的觸發）

**目標**：讓已有 podcast 習慣的人（Kevin、冠廷）找到「對到我工作場景」的一集，產生「這比我現在聽的更值得那個時間位置」的判斷。對抗「主題跟我脫節（旅遊點餐）」與「切換成本高、要打敗清單裡現有節目」。

**步驟**
1. 進「探索」→ 看到主題（科技 / 商業 / 文化 / 科學）與職場場景（會議插話 / 委婉拒絕 / 技術對接 / 客戶 small talk）兩種入口。
2. 選「職場場景 → 會議插話」（佳穎要的）或「科技 → 半導體相關」（冠廷要的）。
3. 列出該場景下的集數，每集一行「這集帶你熟什麼」。
4. 試聽一集 → 若命中，加入「想聽」或直接播。
5. 若是 Free 用戶碰到 Pro 集 → 見 Flow 4。

**關鍵決策點**
- **主題分類用「學科」還是「使用場景」？** → 兩者都給，但**場景優先呈現**。因為付費動機是「學了會用到」，場景（會議插話）比學科（商業英語）更貼近「下次開口的時刻」。
- **要不要讓用戶自選難度來篩？** → 不要丟篩選器給他。系統依 B1 錨點給，避免重演「YouTube 找一小時還是放棄」。
- **這裡是回答 Kevin「你跟 All Ears English 差在哪」的唯一機會**：差異要可見——主題對到工作場景 + 難度剛好 + 聽完留得住（連到 Flow 2 的生字本）。

---

## Flow 4：回歸 + 訂閱決策（無罪惡感回來，理性升級）

**目標**：讓中斷數週的人回來不被責備、讓有付費意願的人在「確定自己會持續用」後才升級。對抗宜君「斷掉重啟罪惡感高」與 Kevin「怕付了不用 / 怕忘記取消」。

**步驟**
1. 隔了 3 週再打開 App → 落地「今天」，看到的是「今天這集」，**不是**「你已 21 天沒來」。
2. 想看自己累積了什麼 → 進「我的收穫」，看到生字本 + 聽過的集（中性語氣，無 streak 斷裂提示）。
3. 連續用了一陣、碰到 Pro 集 → 升級頁清楚說明「Pro 解鎖什麼」（更多場景 / 離線下載 / 完整逐字稿）。
4. 訂閱頁把「取消方式」寫在顯眼處，不藏。
5. 完成升級 → 回到原本想聽的那集，無縫續聽。

**關鍵決策點**
- **回歸時第一眼給什麼？** → 給「今天這集」，絕不給「你斷了 N 天」。這是宜君流失模式的直接解藥：回來零愧疚。
- **何時才提訂閱？** → 在用戶已經感受到價值（聽過幾集、查過詞、看過收穫）之後，而非首聽就擋牆。Kevin 的閾值是「確定自己會持續用」，所以付費牆要晚、要軟。
- **取消入口藏不藏？** → 明確不藏。把「怕忘記取消被一直扣錢」的焦慮做成信任資產：好取消＝敢訂閱。
